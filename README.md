# Herb-recognition
中草药识别系统

## 新增能力（TensorFlow.js 训练）
- 首页新增 `模型训练` 入口（进入训练工作台）
- 支持两种训练样本导入方式：
- 单类打标上传：先选药材类别，再多选图片上传
- 目录批量导入：按 `类别文件夹/图片` 结构导入
- 支持浏览器端训练（Epoch/Batch 参数）、训练进度可视化
- 支持模型保存到浏览器 IndexedDB（下次可加载）

## 数据集调研结果
- 见 `docs/tcm-datasets.md`

## 服务端训练模式
- 进入 `模型训练` 页面后，可配置训练 API 地址（默认 `http://127.0.0.1:4000/api`）
- 页面会把地址保存到浏览器 `localStorage.herbApiBaseUrl`
- 样本上传、样本管理、训练任务提交和状态展示均通过 Nest API 完成
